## Installation

TNLiveDetect is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
1. pod 'TNLiveDetect'
2. target build setting 将 bitcode 设置为 NO
3. 在Info.plist中打开相机访问权限

```
